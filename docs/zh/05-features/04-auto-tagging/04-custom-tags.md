# K8s Label

DeepFlow 目前支持自动关联 K8s 自定义 Label 的资源包括：
- 容器服务
- 工作负载
  - Deployment
  - StatefulSet
  - DaemonSet
  - ReplicationController
  - CafeDeployment
  - CloneSet
- ReplicaSet/InPlaceSet
- Pod

# K8s Annotation

DeepFlow（仅企业版）目前支持自动关联 K8s 自定义 Annotation 的资源包括：
- 容器服务
- Pod

# K8s Env

DeepFlow（仅企业版）目前支持自动关联 K8s 自定义 Annotation 的资源包括：
- Pod

# 云资源自定义标签

DeepFlow（仅企业版）目前支持自动关联云资源自定义标签的资源包括：
- 云服务器

支持的公有云厂商包括：
- Aliyun 阿里云
- Tencent Cloud 腾讯云
