# 流日志

流日志是按分钟粒度记录每一条流的详细信息，通过趋势分析图及表格的形式展示流日志数据。通过在链路上的采集点获取到的数据层层解析后进行 AutoTagging 自动打标签后对数据进行梳理后得到流日志。
