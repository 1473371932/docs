# 概述

DeepFlow 网络模块拥有丰富的功能支持用户对网络中的路径流量及网络性能进行实时监控，包含了资源节点信息、流日志数据、NAT 追踪、PCAP 下载、流量分发、资源盘点等功能。实现了对传输的数据流量、延迟、丢包率等性能参数进行实时监控、分析和评估，可以及时发现和解决网络拥塞、故障、安全等问题，提高网络的稳定性和可靠性，保证网络的高效运转。
